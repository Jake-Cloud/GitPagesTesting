Goals

The main goal of this project is to create an educational website to teach students and professionals the basics of cybersecurity. We hope to accomplish by providing two services. The first is an website that users can find information about a number of different attacks and vulnerabilites. The second is a web server in a docker container that can be run locally by users to practice different penetration testing techniques, and see how exploits would be used against an actual website.
